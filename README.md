#### 4일차
- HTML 태그성질 학습
- 태그 성질에 따른 분류
    - 블록 요소 태그
        - html, body, div, p, ul, ol, li, dl, dt ...
        - 기본 가로 공간의 크기가 부모태그와 동일하게 인식
        - 연속해서 작성하면 세로로 배열됨
        - 공간값이 적용됨(w, h)
    - 인라인 요소 태그
        - a, span, strong, i, em ...
        - 기본 가로 공간의 크기가 안쪽에 포함된 내용만큼만 인식(CONTENT)
        - 연속해서 작성하면 가로로 배열됨
        - 공간값이 적용 안됨
    - 인라인 블록요소 태그
        - img, input, button ...
        - 기본 가로 공간의 크기가 안쪽에 포함된 내용만큼 인식
        - 연속해서 작성하면 가로배열
        - 공간값이 적용됨
